## How to Use These Prompts

1. **Visit Google ImageFX**: https://labs.google/imagefix/
2. **Copy one prompt** at a time into the text field
3. **Click "Generate"** and wait for the AI to create the image
4. **Download** the generated images
5. **Optimize** for web (compress with TinyPNG or similar)
6. **Upload to** `/client/public/images/hero/`

---

## Customization Tips

- **Change colors**: Add "with blue and silver color scheme" to match your brand
- **Add text overlay**: Mention "suitable for text overlay" at the end
- **Change mood**: Swap "professional" for "energetic", "modern", "premium", etc.
- **Adjust subjects**: Replace "sports car" with specific vehicle types
- **Brand integration**: Add "[Your Brand Name]" shop logos if desired

## SEO & Performance

When adding images to the hero:
1. Use descriptive filenames: `hero-premium-auto-parts.jpg`
2. Add alt text: `"Premium automotive parts for performance vehicles"`
3. Compress images to < 200KB each
4. Use WebP format if possible for better compression
5. Add responsive `srcset` for mobile/tablet versions
